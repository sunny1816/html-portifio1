# Personal Portfolio Website

A modern and responsive portfolio website showcasing my skills, projects, achievements, and contact information.

## 🚀 Live Demo

Visit the live website:
[Portfolio Website](https://sunny-capstone.netlify.app/)

## 📌 Features

* Responsive design for all devices
* Modern and clean user interface
* About Me section
* Skills showcase
* Projects section
* Contact form
* Smooth navigation and animations
* Fast loading performance

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Netlify (Deployment)

portfolio/

## 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/sunny1816/html-portifio1.git
```

2. Open the project folder:

```bash
cd html-portifio1
```

3. Open `index.html` in your browser.
